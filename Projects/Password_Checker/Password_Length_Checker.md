# Password Length Checker (Python)

## Description

This project is a beginner-friendly Python script that checks the length of a user-entered password and provides simple feedback based on the number of characters entered.

The program demonstrates how conditional statements can be used to validate user input and categorize password length.

## Purpose

I created this project to practice:

* User input (`input()`)
* The `len()` function
* Conditional statements (`if`, `elif`, `else`)
* Basic input validation
* Decision-making in Python

## Features

* Prompts the user to enter a password.
* Checks the length of the password.
* Displays feedback based on the password length.
* Demonstrates the use of Python's `len()` function.

## Requirements

* Python 3.x

No external libraries are required.

## How to Use

1. Open the Python script.
2. Run the program.
3. Enter a password when prompted.
4. The program will evaluate the password length and display one of the following messages:

   * **Password is too short**
   * **Password is near**
   * **Strong Password**

## Note

This project checks **only the length** of a password. It does **not** evaluate password complexity, such as uppercase letters, lowercase letters, numbers, or special characters. It is intended as a beginner learning exercise.
