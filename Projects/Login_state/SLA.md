# Simple Login Authentication (Python)

## Description

This project is a beginner-friendly Python script that demonstrates how a basic username and password authentication system works using conditional statements.

The program compares user input against predefined credentials and displays whether access is granted or denied.

## Purpose

I created this project to practice:

* Variables
* User input (`input()`)
* Conditional statements (`if` / `else`)
* Logical operators (`and`)
* Basic authentication concepts

## Features

* Prompts the user for a username and password.
* Verifies the entered credentials against stored values.
* Displays either **ACCESS GRANTED** or **ACCESS DENIED**.

## Requirements

* Python 3.x

No external libraries are required.

## How to Use

1. Open the script in Python.
2. Run the program.
3. When prompted, enter the username and password.
4. If both match the predefined credentials in the script, access will be granted.
5. If either value is incorrect, access will be denied.

## Note

This project is intended for learning purposes only. The username and password are stored directly in the source code, which is **not** a secure method for real applications. Production authentication systems should use secure password hashing, encrypted communication, and proper user management.
