# Level 1 Virus Scanner

## Description

This project is a beginner-friendly Python program that demonstrates the basic concept of signature-based malware detection.

The scanner compares a user-provided filename against a small list of predefined example threat names. If a match is found, the program reports that a known threat has been detected. Otherwise, it reports that no known threats were found.

## Purpose

I created this project to practice:

* Lists
* User input
* Conditional statements
* Membership testing (`in`)
* Basic cybersecurity concepts
* Writing simple detection logic in Python

## Features

* Accepts a filename from the user.
* Compares the filename against a predefined list of example threat names.
* Reports whether a matching threat is detected.

## Requirements

* Python 3.x

No external libraries are required.

## How to Use

1. Open the Python script.
2. Run the program.
3. Enter the name of a file when prompted.
4. The program checks whether the filename matches one of the predefined example threat names.
5. A message is displayed indicating whether a known threat was detected.

## Note

This project is an educational demonstration of basic signature-based detection. It does **not** analyze files, inspect file contents, calculate hashes, or provide real antivirus protection. It is intended to help beginners understand the basic concept behind signature matching.
